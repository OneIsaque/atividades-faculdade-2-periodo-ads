
# Atividade de Programação: O Poder da Música 🎵

Este projeto em Java simula um cenário onde a música transforma o estado emocional de uma personagem. A estrutura foi baseada no modelo UML desenvolvido para a atividade.

## 📋 Sobre o Projeto

O objetivo é demonstrar os conceitos de Programação Orientada a Objetos (POO):

 - Classes e Objetos: Representação de entidades do mundo real.
 - Atributos e Métodos: Características e ações de cada classe.
- Relacionamentos: Como as classes interagem entre si.

## 🏗️ Estrutura do Modelo UML

O sistema é composto por 5 classes principais:

- Menina: A protagonista que reage às músicas.

- Musica: A entidade que contém os dados da canção (Título, Artista, BPM).

- Playlist: Agrupador de músicas que permite a navegação.

- Instrumento: Fonte sonora que influencia o ambiente.

- Ambiente: Define o local onde a música acontece (Quinta classe adicionada).

## 🚀 Como Executar

- Certifique-se de ter o JDK instalado.

- No VS Code, abra a pasta do projeto.

- Certifique-se de que os arquivos estão dentro do pacote musicasalva.

- Abra o arquivo App.java e clique em Run.

## 🛠️ Tecnologias Utilizadas

- Linguagem: Java 17+

- IDE: Visual Studio Code
